# Python Exercises

This repository contains my solutions to various Python programming exercises. It is designed to track my progress in mastering Python, covering core concepts and problem-solving techniques.

## Covered Topics
* **Fundamentals:** Variables, loops, and conditional statements.
* **Collections:** Lists, dictionaries, tuples, and sets.
* **Functional Programming:** Lambda functions, map, filter, and decorators.
* **File I/O:** Reading and writing files.
* **API & Data:** Basic scripts for data manipulation.
